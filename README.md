# AYNEXT - AI Smart Glasses Assistant

AYNEXT is an AI-powered smart glasses assistant designed to support users through voice interaction, scene understanding, Arabic-first responses, and contextual recommendations.

The project combines AI, speech processing, backend reasoning, and user-context modeling to create a practical assistive system that can understand the user's situation and provide helpful responses in real time.

## Key Features

- Arabic-first AI assistant responses
- Voice interaction pipeline
- Speech-to-text using faster-whisper
- Text-to-speech support
- Python FastAPI backend
- Local LLM integration using Ollama
- SQLite database for user profiles and context
- Scene/context-aware assistant responses
- Social recommendation prototype
- Daily user-state dashboard concept

## Project Architecture

The system is built around a laptop-first demo architecture:

1. The user speaks to the assistant.
2. The voice pipeline captures and transcribes the speech.
3. The backend receives the user request with optional scene/context data.
4. The AI reasoning layer generates an Arabic-first response.
5. The system returns a useful response to the user through text or voice.
